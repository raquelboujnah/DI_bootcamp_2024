Event Breakdown: mousedown – Triggers when the mouse button is pressed down. This will start the coloring. mouseup – Triggers when the mouse button is released. This will stop the coloring. mouseover – Triggers when the mouse moves over an element. This will color the cells, but only if the mouse button is already pressed. How It Works: You'll need to create a variable - isMouseDown, witch will be a boolean - false. When you press the mouse button down on a cell (mousedown), it changes color and sets isMouseDown = true. As you move the mouse over other cells (mouseover), if isMouseDown is true, those cells will also be colored. When you release the mouse (mouseup), isMouseDown is set to false, stopping the coloring effect.
